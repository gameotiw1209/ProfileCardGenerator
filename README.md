# 🎴 ProfileForge – Profile Card Generator

ProfileForge is a simple and interactive web application that allows users to create a stylish **profile card** by filling out a form.  
The card is generated dynamically and can be **downloaded as an image**.

This project is built using **HTML, CSS, and JavaScript**.

---

## 🚀 Features

✨ Create a personalized profile card  
✨ Add profile image, name, job title, and hobbies  
✨ Include social media links (Instagram, WhatsApp, GitHub, X, Telegram)  
✨ Beautiful animated gradient background  
✨ Interactive hover animations  
✨ Download the generated card as an image  
✨ Fully responsive design

---

## 🛠 Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **Font Awesome Icons**
- **html2canvas** (for downloading the card as an image)

## ⚙️ How It Works

1. User opens the **form page**
2. Enters profile details like:
   - Name
   - Job title
   - Hobbies
   - Profile image
   - Social media links
3. Clicks **Generate Card**
4. The data is stored using **LocalStorage**
5. The user is redirected to the **Profile Card Page**
6. The card is generated dynamically with the entered information
